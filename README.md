# 📊 Sistema de Médias em C++

Programa simples que cadastra alunos, recebe 3 notas, calcula a média e diz se o cara passou ou rodou.

## 💻 O que faz

* Lê nome do aluno (inclusive nome composto)
* Lê 3 notas com validação (0 a 10)
* Calcula média
* Mostra tudo em formato de tabela
* Diz se tá Aprovado ou Reprovado
* Mostra a maior média da turma

## ▶️ Como rodar

```bash
g++ main.cpp -o medias
./medias
```

Só um projetinho pro Debatin não mandar minha nota pro espaço.
